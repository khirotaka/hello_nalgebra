# nalgebra 入門

Rustで数値計算するためのライブラリ入門

## System Requirements
* Apple M1
* Rust 1.53.0


## 分かったこと
BLASとリンクしていないのか、Apple AMXが動かないようで、行列積が遅い。


## 実行例

```shell
$ cargo build --relase
$ ./target/release/hello_nalgebra
```
